#房貸計算器

功能:輸入房價、自備款比例、貸款年限、貸款寬限期、年利率，按下計算後，就能算出需要貸款的金額、總利息支出、總還款金額和每月應繳的金額等等。讓大家在計算貸款金額時可以更方便。
